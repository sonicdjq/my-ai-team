# 家庭财务管家App - 产品需求文档

## 1. 产品概述

家庭财务管家App是一款面向个人和家庭的财务管理应用，旨在帮助用户记录收支、管理账户资产、设置预算并通过报表分析财务数据。

### 核心目标
- 帮助用户清晰了解个人/家庭的收入、支出和财务健康状况
- 提供便捷的记账体验，降低财务管理的门槛
- 通过数据可视化和分析，为用户提供有价值的财务洞察
- 支持多用户协作，满足家庭共同财务管理的需求

### 目标用户
- 个人用户：需要管理个人财务的成年人
- 家庭用户：需要共同管理家庭财务的家庭成员

### 产品价值
- 提升用户的财务管理意识和能力
- 帮助用户养成良好的消费习惯
- 为家庭财务决策提供数据支持
- 减少财务管理的时间和精力成本

## 2. 功能需求

### 2.1 交易管理（核心功能）

#### 新增交易
- 记录收入、支出和转账交易
- 支持的字段：金额、类型、分类、账户、日期、商户（可选）、标签（可选）、备注（可选）

#### 编辑交易
- 修改现有交易的所有字段

#### 删除交易
- 移除不需要的交易记录

#### 交易列表
- 查看交易历史，默认按日期排序
- 支持滑动操作进行快速编辑/删除

#### 交易搜索与筛选
- 支持按日期范围、分类、账户、交易类型进行筛选
- 支持关键词搜索

### 2.2 分类管理

#### 默认分类
- 支出分类：餐饮、交通、购物、娱乐、住房、医疗等
- 收入分类：工资、奖金、投资、礼金等

#### 自定义分类
- 用户可创建、编辑、删除自定义分类
- 支持分类的层级结构（父分类和子分类）

### 2.3 账户管理

#### 账户类型
- 支持现金、银行账户、信用卡、电子钱包、其他等账户类型

#### 账户操作
- 创建、编辑、删除账户
- 查看账户余额

#### 账户转账
- 支持账户间的资金转账
- 转账字段：转出账户、转入账户、金额、日期、备注

### 2.4 预算管理

#### 月度总预算
- 设置每月总预算
- 实时追踪预算使用进度

#### 分类预算
- 为特定分类设置预算
- 监控分类支出情况

#### 预算提醒
- 当预算使用超过阈值（如80%）时发送提醒
- 当预算超支时发送警告

### 2.5 报表与分析

#### 支出分析
- 按分类展示支出情况
- 支持饼图、柱状图等可视化方式

#### 时间维度分析
- 按日、周、月、年查看财务数据

#### 收支对比
- 展示总收入、总支出和净余额
- 支持不同时期的对比分析

### 2.6 账本管理

#### 多账本支持
- 用户可创建多个账本（如个人账本、家庭账本、旅行账本、项目账本等）
- 交易数据与特定账本关联

#### 账本操作
- 创建、切换、编辑、删除账本

### 2.7 数据管理

#### 数据导出
- 支持导出为CSV、Excel格式

#### 数据备份
- 支持本地备份和云端同步

#### 数据导入
- 支持从CSV、Excel导入交易数据

### 2.8 用户系统

#### 认证功能
- 支持邮箱登录、社交登录
- 支持密码重置

#### 安全特性
- 加密存储用户数据
- 支持生物识别解锁（指纹/面容ID）
- 提供隐私保护设置

### 2.9 多用户协作（家庭模式）

#### 协作功能
- 邀请家庭成员加入账本
- 分配不同权限级别
- 查看共享交易记录
- 支持协作记账

### 2.10 高级功能（可选）

#### 智能记账
- 自动银行交易导入
- 支付平台集成
- AI分类预测

#### 快速录入
- 语音输入
-  receipts照片识别
- 常用模板

#### 通知提醒
- 预算警报
- 异常支出提醒
- 定期账单提醒

## 3. 非功能需求

### 3.1 性能
- 交易创建响应时间 < 200ms
- 报表加载时间 < 1秒
- 应用启动时间 < 3秒

### 3.2 安全性
- 用户数据加密存储
- 敏感信息传输加密
- 定期安全审计

### 3.3 可扩展性
- 支持每个用户创建多个账本
- 支持大量交易数据的存储和查询
- 模块化设计，便于功能扩展

### 3.4 可用性
- 支持离线记账，网络恢复后自动同步
- 界面简洁直观，操作流程简单
- 提供详细的使用帮助和教程

### 3.5 兼容性
- 支持Web端访问
- 适配不同屏幕尺寸和浏览器
- 响应式设计，支持桌面和移动设备的Web访问

## 4. 数据模型

### 核心实体
- 用户 (User)
- 账本 (Ledger)
- 账户 (Account)
- 交易 (Transaction)
- 分类 (Category)
- 预算 (Budget)
- 标签 (Tag)

### 关系
- 用户 -> 账本（一对多）
- 账本 -> 账户（一对多）
- 账本 -> 交易（一对多）
- 交易 -> 分类（多对一）
- 交易 -> 账户（多对一）
- 分类 -> 预算（多对一）

## 5. 用户流程

### 5.1 注册/登录流程
1. 用户打开应用
2. 选择注册或登录
3. 输入注册信息或登录凭证
4. 完成身份验证
5. 进入应用主界面

### 5.2 记账流程
1. 用户点击"新增交易"按钮
2. 选择交易类型（收入/支出/转账）
3. 输入金额
4. 选择分类
5. 选择账户
6. 填写其他可选信息（商户、标签、备注）
7. 确认并保存

### 5.3 查看报表流程
1. 用户进入"报表"页面
2. 选择时间范围
3. 选择报表类型（支出分析、收支对比等）
4. 查看可视化数据
5. 可选择导出报表

### 5.4 预算设置流程
1. 用户进入"预算"页面
2. 设置月度总预算
3. 为特定分类设置预算
4. 保存预算设置
5. 查看预算执行情况

### 5.5 多用户协作流程
1. 账本创建者进入账本设置
2. 邀请家庭成员
3. 分配权限
4. 家庭成员接受邀请
5. 开始协作记账
